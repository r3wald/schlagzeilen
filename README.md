# Schlagzeilen-Generator

## Installation
1. git clone https://github.com/r3wald/schlagzeilen
2. cd phrasen
3. composer install
4. php -S 0.0.0.0:12345 -t htdocs/
5. xdg-open http://localhost:12345/
